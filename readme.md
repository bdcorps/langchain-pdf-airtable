

This repo lets you ask 10-Q PDFs ask questions > Save the answers as markdown in an Airtable base

## How to use

1. Add a .env file:

```
OPENAI_API_KEY=
AIRTABLE_API_KEY=
AIRTABLE_BASE_ID=
```

2. Run `npm i`
3. Run `nm run dev`

You will see a response in your console for the Microsoft 10-K report
